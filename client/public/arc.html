Data Flow Process

  1. User Interaction: User interacts with React frontend
  2. API Request: Frontend sends HTTP request to Node.js server
  3. Server Processing: Express server processes the request
  4. ML Processing: Node.js calls Python ML server (if ML required)
  5. ML Response: Python server returns processed data to Node.js
  6. Database Operation: Node.js performs database operations
  7. Response: Node.js sends final response back to React frontend
  8. UI Update: React updates the user interface
